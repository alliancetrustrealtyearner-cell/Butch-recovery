Place your static assets (logos, icons) in this folder.

For example:
- gem-logo.svg (downloaded from https://upload.wikimedia.org/wikipedia/commons/4/4a/Commons-logo.svg)
- bitcoin-logo.svg (downloaded from https://upload.wikimedia.org/wikipedia/commons/4/46/Bitcoin.svg)

Ensure the references in index.html use the local files from assets/ for proper GitHub Pages deployment.
